Sección de Fórmulas

En esta sección encontrarás fórmulas matemáticas importantes:

  • Rectángulo `A=bxxh` `b` : base `h` : altura
  • Triángulo `A=(bxxh)/2` `b` : base `h` : altura
  • Rombo `A=(Dxxd)/2` `D` : diagonale mayor `d` : diagonal menor
  • Trapecio `A=(B+b)/2xxh` `B` : base mayor `b` : base menor `h`: altura

Sección de Ejercicios

Resuelve los siguientes ejercicios:

  1. Hallar el area de un rectangulo que tiene una base de 10cm y una altura de 50cm
  2. Hallar el area de un trapecio cuyas bases son de 3 y 5 cm y altura es de 10cm
  3. Hallar area de un triangulo que tiene 3m de largo y 5m de alto
